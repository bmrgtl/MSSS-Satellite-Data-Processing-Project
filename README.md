**OVERVIEW**

This application is developed as part of the SM TAFE Complex Data Structures Cluster for Malin Space Science Systems (MSSS) to replace their outdated 32-bit command line interface with a modern C# WPF application GUI that loads, sorts, and searches data.

**FEATURES**
1. Load dataset using Galileo.DLL
2. Implements data validation for sigma (Min: 10, Max: 20, Default: 10) and mean (Min: 35, Max: 75, Default: 50) values
3. Binary Search Iterative and Recursive Algorithms
4. Selection and Insertion Sort Algorithms
5. System Notification and Error Handling
